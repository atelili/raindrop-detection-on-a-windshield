# raindrop-detection-on-a-windshield

Raindrop detection on a windshield using image processing.

## Requirements
opencv<br/>
statistics<br/>
argparse<br/>
timeit<br/>


## Usage

```bash
python raindrop_detection.py -i image_path
```
